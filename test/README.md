# Axisymmetric Stokes close-evaluation — Green's representation test

`test_axissymstok_GRF.m` validates the per-panel azimuthal-mode **close-evaluation** for the
axisymmetric Stokes single- and double-layer potentials, end to end, against an exact field.

`axissymstok_kernelsplit_mod.f90` is hidden for now. Need more future tests...

## What it checks

For a Stokes flow `u` generated by a point force, the Green's representation formula

```
u(x) = S[σ](x) + D[τ](x),     σ = traction(u),   τ = -u   on the surface
```

reproduces `u(x)` for `x` inside the body and **exactly 0** for `x` outside it. The test uses a
single **off-axis Stokeslet** as the manufactured field, so the surface densities are *not*
axisymmetric and carry every azimuthal mode. The densities are sampled at 64 azimuths, FFT'd in
`θ`, truncated to modes `|m| ≤ 8`, and the field is reconstructed on a meridian (`θ = 0`) grid.

Targets are restricted to the **exterior**, where the representation is identically zero
mode-by-mode (no mode-truncation floor) — this isolates the close-evaluation accuracy near the
surface, where a plain quadrature breaks down.

The script prints two subsections:

- **Subsection 1 — naive kernels.** Smooth Gauss quadrature with the bare modal kernels. Accurate
  far from the surface, but fails in the near band (error `~1e-1`).
- **Subsection 2 — kernel-split close-eval.** The near-target correction built from the modal
  split coefficients + Helsing singular quadrature. The near band collapses to the close-eval
  floor.

## Running

With the compiled mex on the MATLAB path (and the `lege` Legendre routines available), run:

```matlab
test_axissymstok_GRF
```

Expected console output (torus tube, modes 0..8, exterior targets only):

```
  SUBSECTION 1 (naive):       exterior far  (>0.3)  max err = 6.3e-14
  SUBSECTION 1 (naive):       exterior near (<0.05) max err = 8.4e-02   (naive quadrature fails)
  SUBSECTION 2 (Fortran mex): exterior near (<0.1)  max err = 7.6e-12   (close-eval floor, modes 0..8)
  SUBSECTION 2 (Fortran mex): exterior far  (>0.3)  max err = 6.2e-12
```
